# Cheap Browser

Work in progress  

## Run  
```
Python 3.10.5 ( Should work with Python 3+)
pip install -r requirements.txt
python main.py
```
### For Linux Users
```
sudo pacman -S tk                   # Arch Based Distros
sudo apt-get install python3-tk     # Debian based distros
sudo dnf install -y python3-tkinter # Fedora
```
